# Academic Homepage for GitHub Pages

This is Jingwen Zhang's lightweight static academic homepage, inspired by
[al-folio](https://github.com/alshedivat/al-folio) and laid out similarly to
Xiao Li's academic homepage.

## Edit the site

- Update personal information in `index.html`.
- Replace `assets/img/profile-placeholder.svg` with your own photo, then update
  the image path in `index.html` if the filename changes.
- Put your CV at `assets/cv.pdf`, or change the CV link in `index.html`.
- Add publications and projects in their matching sections.
- The top navigation is intentionally minimal and contains only `Home`.

## Deploy to GitHub Pages

1. Create a repository named `your-github-username.github.io`.
2. Upload these files to the repository root.
3. In GitHub, open `Settings > Pages`.
4. Set the source to `Deploy from a branch`, branch `main`, folder `/root`.
5. Visit `https://your-github-username.github.io/` after GitHub finishes deployment.

No build step is required.
